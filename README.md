# Health Insurance Claim Prediction — MCF ITB x AXA Financial

Regression model to forecast health insurance claim amounts. Built for the Machine Learning competition hosted by MCF ITB in collaboration with AXA Financial.

## Approach

- Exploratory data analysis on claim patterns and policyholder features
- Feature engineering: interaction terms, binning, encoding categorical variables
- Trained Ridge Regression as the primary model
- Evaluated using RMSE and MAE metrics

## Tech Stack

- Python, Scikit-learn
- Pandas, NumPy, Matplotlib, Seaborn

## Results

Submitted predictions for the MCF ITB x AXA Financial competition leaderboard.

## Files

- `claim_prediction.ipynb` — Full EDA + modeling pipeline
- `submission.csv` — Final competition submission
